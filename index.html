These OpenApi document files are automatically generated and copied into the /docs folder during CI/CD.

Then, the documents and this html page are published in GitHub Pages so that this url is publicly available.

Finally, the CI/CD pipeline triggers a repository dispatch with the event update-sdk so that an SDK repository can automatically initiate its CI/CD pipeline.
